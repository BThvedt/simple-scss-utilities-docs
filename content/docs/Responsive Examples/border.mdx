# Border

#### How to use

Like margin and padding, border and outline can be applied in any direction but there's less classes. You can find the widths in `_variables.scss` and like with everything else both the keys and the values can be edited, added, or removed:

<CodeWrapper title="_variables.scss_" isFile >

```scss
$border-and-outline-widths: (
  "0": 0px,
  "sm": 1px,
  "md": 3px,
  "lg": 5px,
  "xl": 7px,
  "2xl": 11px
);
```

</CodeWrapper>

These default values generate default classes like you would expect.

| Class      | CSS rule            |
| :--------- | :------------------ |
| border-0   | border-width: 0px;  |
| border-sm  | border-width: 1px;  |
| border-md  | border-width: 3px;  |
| border-lg  | border-width: 5px;  |
| border-xl  | border-width: 7px;  |
| border-2xl | border-width: 11px; |

Border classes include [Colors](/docs/classes-with-colors/?anchor=border-color) or [Styles](/docs/border/?anchor=top-right-bottom-left) which are covered in their own pages, but it's pretty simple and I'll include those in the examples below.

In addition to these, there's also classes for the left, right, top, and bottom borders which are `.bl`, `.br`, `.bt`, and `.bb` classes respectively. There are also x and y classes `.bx` and `.by`:

| Class  | CSS rule                                               |
| :----- | :----------------------------------------------------- |
| bl-0   | border-left-width: 0px;                                |
| bl-sm  | border-left-width: 1px;                                |
| bl-md  | border-left-width: 3px;                                |
| bl-lg  | border-left-width: 5px;                                |
| bl-xl  | border-left-width: 7px;                                |
| bl-2xl | border-left-width: 11px;                               |
| br-0   | border-right-width: 0px;                               |
| br-sm  | border-right-width: 1px;                               |
| br-md  | border-right-width: 3px;                               |
| br-lg  | border-right-width: 5px;                               |
| br-xl  | border-right-width: 7px;                               |
| br-2xl | border-right-width: 11px;                              |
| bt-0   | border-top-width: 0px;                                 |
| bt-sm  | border-top-width: 1px;                                 |
| bt-md  | border-top-width: 3px;                                 |
| bt-lg  | border-top-width: 5px;                                 |
| bt-xl  | border-top-width: 7px;                                 |
| bt-xl  | border-top-width: 11px;                                |
| bb-0   | border-bottom-width: 0px;                              |
| bb-sm  | border-bottom-width: 1px;                              |
| bb-md  | border-bottom-width: 3px;                              |
| bb-lg  | border-bottom-width: 5px;                              |
| bb-xl  | border-bottom-width: 7px;                              |
| bb-2xl | border-bottom-width: 11px;                             |
| bx-0   | border-left-width: 0px;<br/>border-right-width: 0px;   |
| bx-sm  | border-left-width: 1px;<br/>border-right-width: 1px;   |
| bx-md  | border-left-width: 3px;<br/>border-right-width: 3px;   |
| bx-lg  | border-left-width: 5px;<br/>border-right-width: 5px;   |
| bx-xl  | border-left-width: 7px;<br/>border-right-width: 7px;   |
| bx-2xl | border-left-width: 11px;<br/>border-right-width: 11px; |
| by-0   | border-top-width: 0px;<br/>border-bottom-width: 0px;   |
| by-sm  | border-top-width: 1px;<br/>border-bottom-width: 1px;   |
| by-md  | border-top-width: 3px;<br/>border-bottom-width: 3px;   |
| by-lg  | border-top-width: 5px;<br/>border-bottom-width: 5px;   |
| by-xl  | border-top-width: 7px;<br/>border-bottom-width: 7px;   |
| by-2xl | border-top-width: 11px;<br/>border-bottom-width: 11px; |

### Responsive Examples

Like margin and padding, border width classes can be prefixed with the breakpoints defined in `_variables.scss`. Here is an example of the border changing sizes from small to large widths. If you're on a browser you can resize the screen to see the border width, on the smallest widths you'll have to close the sideber. I didn't make order radius, style classes don't respond to breakpoints because they don't have much to do with size, if you need those to be responsive it's a good opportunity to write some custom SCSS! But border size is at least responsive:

<div className="center m-lg mb-xl flex-col my-xl">
  <div className="w-5 h-5 bg-blue radius-md border-md tab:border-lg scr:border-xl lg:border-2xl " />
  <Paragraph className="display-none max-mob:display-block" text=".border-md" />
  <Paragraph
    className="display-none tab:display-block scr:display-none"
    text=".tab:border-lg"
  />
  <Paragraph
    className="display-none scr:display-block lg:display-none"
    text=".scr:border-xl"
  />
  <Paragraph className="display-none lg:display-block" text=".lg:border-2xl" />
</div>

Here is the code for the responsive border size:

<CodeWrapper title="HTML" >

```html
<div
  class="w-5 h-5 bg-blue radius-md border-md tab:border-lg scr:border-xl lg:border-2xl "
/>
```

</CodeWrapper>

And as one more example, here's a border with a `.max-scr` class, so the outline is `lg` up to 1024px (the break point between `scr` and `lg`) and `2xl` above that:

    className="display-none max-scr:display-block"
    text=".max-scr:border-lg"
  />
  <Paragraph className="display-none lg:display-block" text=".border-2xl" />
</div>

And here's the HTML:


</CodeWrapper>
